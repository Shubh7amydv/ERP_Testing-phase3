## API Integration Points

| Screen | API Calls |
|--------|-----------|
| Staff List | `GET /staff/` |
| Add Staff | `POST /staff/` |
| Staff Detail | `GET /staff/{id}/` |
| Staff Attendance | `GET /staff/{id}/attendance/` |
| Leave Types | `GET /leave-types/` |
| Apply Leave | `POST /staff-leaves/` |
| Approve Leave | `POST /staff-leaves/{id}/approve/` |
| Leave Balance | `GET /staff-leaves/balance/{staff_id}/` |
| Payroll Months | `GET /payroll-months/` |
| Process Payroll | `POST /payroll-months/{id}/process/` |
| Generate Slips | `POST /salary-slips/generate/` |
| Download Slip | `GET /salary-slips/{id}/download/` |
| Salary Components | `GET /salary-components/` |

---

## Key Components

| Component | Type | Description |
|-----------|------|-------------|
| `HRDashboard` | Page | Staff stats + pending actions |
| `StaffDirectory` | Page | Searchable staff list |
| `StaffProfile` | Page | Tabbed staff detail |
| `StaffForm` | MultiStep | Add/edit staff |
| `LeaveApplication` | Page | Leave management |
| `PayrollProcessor` | Page | Monthly payroll |
| `SalarySlip` | Printable | Salary slip template |
| `AttendanceTracker` | Page | Staff attendance |
