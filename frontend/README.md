# Dettroin ERP - School Management System

Gridaan ERP is a premium School Management Workspace built using React, TypeScript, and Vanilla CSS. It provides a state-of-the-art administrative portal for managing school operations, admissions, fees, and registers.

## Features

- **Dashboard Insights**: Interactive KPIs, admissions vs. fee collection trend graph, circular gauges, class distribution, and real-time system logs.
- **Student Directory**: Advanced search, filters, pagination, profile avatar views, and dynamic institutional ID card generator.
- **Admission Portal**: Multi-column form fields for students and parent records with active validation.
- **Student Reports**: Dedicated panel sections for Roll lists, House divisions, Sibling grouping, and direct student photo uploads.
- **Fees Collector**: Month-wise billing checklists allowing lump-sum checkouts ("Collect in one hand") and detailed paid transaction history ledger.
- **Gatepass Registry**: Request exit permits during school hours and approve/reject logs.
- **Activity Events**: Coordinate academic calendar dates and coordinate event schedules.
- **Teacher Directory**: Manage faculty statuses and assigned classes.

## Tech Stack

- **Frontend Framework**: React + TypeScript
- **Bundler**: Vite
- **Styling**: Custom CSS (Slate-Navy & Royal Blue Palette)
- **Icons**: Lucide React
- **Database/Backend Schema**: Standalone Frontend (Mock API layer with localStorage persistence)

## Setup and Running Locally

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the local development server:
   ```bash
   npm run dev
   ```
